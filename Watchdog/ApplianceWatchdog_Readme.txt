
Build Instructions: 
1) Appliance Watchdog is an Android Studio project:
   http://developer.android.com/sdk/installing/studio.html
2)Set your Android SDK path using following commands: 
  File -> project Structure into Project Structure
  Left -> SDK Location
  SDK location select Android SDK location (old version use Press  +, add another sdk)
3)Import the project into Android Studio:
  File -> Import Project -> Choose Directory -> Import from external model -> Gradle -> Use default gradle wrapper -> Finish
4)Build -> Clean Project
  Build -> Rebuild Project
5)Enable Android phone for development and connect it to laptop using USB Cable. 
  Steps for enabling android phone for development: 
  5.1) Go to the settings menu, and scroll down to "About phone." Tap it.
  5.2) Scroll down to the bottom again, where you see "Build number." 
  5.3) Tap it seven (7) times. After the third tap, you'll see a playful dialog that says you're four taps away from being a developer. 
6) Once the phone has been connected, click on the play button in android studio and run the app directly on the phone.
